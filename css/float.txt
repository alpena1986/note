子元素是float时,父元素无法感知子元素的height.
至于子元素的width,父元素根本不在意.因为父元素如果不指定width,就是父元素的上级的width的100%.而不会去适应子元素的width.

子元素是float时,父元素感知子元素height的方法.

1. 对父元素使用overflow: hidden;  
    BFC 全称是块级排版上下文，用于对块级元素排版，默认情况下只有根元素（body)一个块级上下文，
    但是如果一个块级元素,设置了float:left，overflow:hidden或position:absolute样式，就会为这个块级元素生成一个独立的块圾上下文.
    独立的块级上下文可以包裹浮动流，全部浮动子元素也不会引起容器高度塌陷，也就是说包含块会把浮动元素的高度也计算在内，所以不用清除浮动来撑起高度。

2. 为子元素增加兄弟子元素,并且对这个兄弟子元素设置clear: both;
    clear可以设定的值包括left ,right ,both等等.
    表示的意思是,该元素的左侧,或者右侧,或者两边不允许出现float元素.
    这样,该元素只能自成一行.由于该元素并非float,所以父元素能够感知到该元素,以达到让父元素感知自己孩子高度的效果.
    
3. 与2类似.但是不用为子元素增加兄弟子元素.为父元素写一个:after类,如下
  .father:after {
    content: '';
    display: table;
    clear: both;
  }
